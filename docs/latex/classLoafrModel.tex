\doxysubsection{Detailed Description}
A model class that encapsulates filtering, searching, and sorting functionalities for log entries. 

This class uses unique pointers to manage \mbox{\hyperlink{classFilter}{Filter}}, \mbox{\hyperlink{classSearch}{Search}}, and \mbox{\hyperlink{classSort}{Sort}} objects, providing a unified interface for processing log file entries.
